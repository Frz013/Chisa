> **IMPORTANT**
> This is not a standalone theme, it requires [HyDE](https://github.com/Hyde-project/hyde) to be installed.

> This theme is an adaptation of the official Tokyo Night theme, which was inspired by the character Chisa from the game Wuthering Waves. In its development, it not only preserves the signature calm and elegant nighttime aesthetic of Tokyo Night theme, but also incorporates visual elements and atmosphere that reflect Chisa’s personality and essence, resulting in a unique color blend of modern, expressive, and artistic style.

## Installation

#### HyDE Project

To install **Chisa** on `HyDE Project`, run the following command:
```sh
hydectl theme import --name "Chisa" --url https://github.com/Frz013/Chisa
```

Alternatively, you can import the theme by running:
```sh
hydectl theme import
```

Then choose **Chisa** from the options.

#### Hyprdots

To install **Chisa** on `Hyprdots`, run the following command:
```sh
Hyde theme import "Chisa" https://github.com/Frz013/Chisa
```

Alternatively, you can import the theme by running:
```sh
Hyde theme import
```

Then choose **Chisa** from the options.

#### General

Alternatively, you can use a general option that works on both `HyDE Project` and `Hyprdots`:
```sh
~/HyDE/Scripts/themepatcher.sh "Chisa" https://github.com/Frz013/Chisa
```
